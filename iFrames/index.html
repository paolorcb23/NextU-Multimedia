ejemplos de iframes y atributos

Ejemplo basico de iFrame

allow-forms: permite que se envien formularios con informacion

allow-popups: si el iFrame tiene ventanas emergentes, este lo permitira

allow scripts, permite que del lado del cliente, se ejecuten codigos javascripts

allow top navigation: permite que la informacion en el iFrame, sustituya la de nosotros, propuesta por el contenido del enlace
