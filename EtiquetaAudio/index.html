Atributo AUTOPLAY

Reporduce el audio o video apenas se abre la web, sin necesidad de dar clic en PLAY

Atributo LOOP

Atributo que se encarga de reproducir el audio, cuando TERMINE

Atributo PRELOAD

este tiene un comportamiento especial. Cuando trabajamos con archivos muy pesados el sitio web se pone lento y demora, este atributo los almacena en una memoria temporal despues de ser cargados la primera vez, depende del valor que se le especifique

  • none: No almacenar nuestros archivos
  • auto: Almacena temporalemente el archivo multimedia.
